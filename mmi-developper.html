My Journey with Web Development

Developing for the web means creating tools, sites, and applications. My goal is to understand client needs and translate them into technical solutions, using programming languages and development tools. I have experience with Windows, Linux, and macOS environments (STI2D background), and have assisted my school's IT technicians with assembling and configuring mini-PCs.

The Project: "The Social Network" Static Site (SAÉ 105)

I built a complete static website dedicated to the film "The Social Network" using HTML5 & CSS3, implementing responsive design with Flexbox. This project mobilised resources in integration (R1.11), web development (R1.12), and hosting (R1.13).

Latest Website Project

Parodiapédia

My latest hand-coded website is a parody recreation of a Wikipedia-style interface called "Parodiapédia". It features a structured navigation with sections like "Sommaire", "Enfance et études", "Vie quotidienne", "Expériences et projets", and "Centres d'intérêt". The text content is partially censored to protect personal information.

Built entirely with HTML5 & CSS3, demonstrating page structure, semantics, and design layout skills.
